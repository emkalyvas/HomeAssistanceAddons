# Degoog

A minimal, modular self-hosted meta-search aggregator that queries multiple search engines from a unified interface.

## Features
- Search across multiple engines cleanly and privately.
- Highly modular with customizable plugins, themes, and extensions.
- Fully integrated into Home Assistant via Ingress (sidebar access).
- Automated persistence for configuration, extensions, and themes.

## Configuration
- **`settings_password`**: (Optional) Password to lock Degoog's settings panel.
