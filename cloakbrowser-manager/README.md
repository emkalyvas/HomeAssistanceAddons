# CloakBrowser Manager

A self-hosted, web-based management interface for creating, configuring, and launching isolated browser profiles with unique fingerprints using CloakBrowser. An open-source alternative to commercial multi-login profile managers.

## Features
- Create and manage isolated browser profiles.
- Custom fingerprint configurations (WebGL, Canvas, screen properties, proxies, etc.).
- Ingress support enabled: Access and interact with browser profiles directly inside your Home Assistant UI.
- Direct interactive access via noVNC.

## Configuration
- **`auth_token`**: (Optional) Add a password token to secure the CloakBrowser Manager API and interface.
